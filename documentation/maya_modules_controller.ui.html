maya_modules_controller.ui¶

Maya Module Controller UI :module: maya_modules_controller.ui

Contains the ui Qt widgets

The UI has a line text edit to type (search/filter) modules as well as a list view widget that will display the modules. A push button widget at the end will be use to trigger the delete modules.
