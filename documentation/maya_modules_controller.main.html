proxy_model¶

Proxy Model is a QSortFilterProxyModel that is applied to the view.

You can use the proxy_model to set some custom behavior to the view.

refresh()[source]¶

Refrehes the modules list.

Refresh will check if a filter has been applied to the tool and refresh the modules list within that filter.

If no filter has been applied to the modules list refresh will query all modules been loaded back from sys.modules
